What are Progressive Web Apps?

Progressive Web Apps (PWAs) are the future of apps. Advancements in web browser technologies have made it possible to create web-based applications that mimic and offer the same functionality as native (traditional) applications. This means that apps can now be made just as easily as websites, and that developers no longer need developer licenses to distribute them.Just like an increasing number of desktop-based applications nowadays offer their functionality online, mobile applications are about to move to the browser as well.
